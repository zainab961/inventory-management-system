# C++based-inventory-management-system
Developed a C++ program to manage inventory, track stock, handle sales and returns, and maintain supplier information. Includes automatic stock alerts and generates sales receipts.
# Inventory Control System
## Overview
This is a **C++ Inventory Control System** for managing products, suppliers, and sales.  
The system allows you to:

- Add products to the inventory  
- Display product details and prices  
- Sell products and generate bills  
- Manage supplier information  
- Track stock availability  
- Handle product returns  

It also automatically alerts when stock is low for any item.

## Features
- Add, display, and manage inventory items  
- Maintain stock levels and alerts for low stock  
- Record sales and generate receipts  
- Track supplier information and incoming stock  
- Handle returns and update inventory automatically  

## Technologies Used
- **C++** – Main programming language  
- File handling (`fstream`) – For storing and reading inventory data  

## How to Run
1. Make sure you have a **C++ compiler** installed (e.g., Dev C++, Visual Studio)  
2. Clone or download the repository  
3. Open the project in your IDE or terminal  
4. Compile the code:

c++ inventory.cpp -o inventory
